# SQL Project Report: Retail Store Analytics

## Project Overview
This project analyzes a retail store database using SQL to understand customer behavior, order patterns, and inventory management.

## Tools & Technologies
- **Database**: MySQL Workbench
- **Concepts Used**: Joins, Subqueries, Aggregations, Group By, Window Functions

## Files Included
- `SQL_Project.zip` - Contains:
    - `Import_Data.sql` - Database schema and sample data
    - `Project_Queries.sql` - All analytical queries
    - `SQL Project Report.pdf` - Complete project report

## Key Analysis Done
1. **Customer Order Distribution** - Buying patterns analysis
2. **Financial Transaction Streams** - Payment method insights
3. **Product Catalog Dependencies** - Product relationship mapping

## Conclusion
This SQL relational analytics model successfully interprets customer order distribution patterns, preferred financial transaction streams, and catalog dependencies. These structured queries provide scalable analytical insights crucial for managing stock replenish cycles and evaluating customer lifetime interactions efficiently.

## How to Run
1. Download and extract `SQL_Project.zip`
2. Open MySQL Workbench
3. Run `Import_Data.sql` to create database
4. Execute queries from `Project_Queries.sql`
